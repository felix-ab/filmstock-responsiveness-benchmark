# Security And Privacy Notes

This repository was prepared for public release with a conservative scrub:

- Provider API keys are not included.
- Local absolute paths were replaced with repo-relative paths in public CSVs and code defaults.
- Generated PNG files were re-saved without ancillary metadata.
- DOCX metadata was normalized.
- Private network information and local machine names were scanned before publication.

To regenerate images, configure credentials locally through environment
variables or your own ComfyUI node setup. Do not commit `.env` files, key files,
or provider credentials.
